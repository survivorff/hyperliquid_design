# Vault 策略与风险：完整图谱

> 除了 HLP，Hyperliquid 上有数百个 Vault，策略各异。这篇梳理主要策略类型、收益逻辑和风险评估框架。

---

## 1. Vault 策略分类

### 1.1 主要策略类型

| 策略 | 收益来源 | 风险 | 历史年化 |
|------|---------|------|---------|
| **做市（Market Making）** | 买卖价差 | 单边敞口 | 5-20% |
| **资金费率套利** | 资金费率收取 | 费率反转 | 10-25% |
| **Delta 中性** | 多空对冲 + 增强 | 执行风险 | 5-15% |
| **趋势跟随** | 方向性押注 | 趋势反转 | 极不稳定 |
| **跟单（Copy）** | 跟随明星交易员 | 交易员失误 | 极不稳定 |
| **统计套利** | 价格回归 | 模型失效 | 10-30% |

### 1.2 风险-收益谱

```
低风险 ←──────────────────────────→ 高风险

Delta中性  资金费率套利  做市  统计套利  趋势跟随  跟单
（稳）                              （博）
```

---

## 2. 做市策略

### 2.1 基本逻辑

```
在盘口同时挂买单和卖单
买单 $59,990，卖单 $60,010（价差 $20）
如果两边都成交 → 赚 $20 价差
```

### 2.2 收益与风险

**收益**：
- 持续赚取价差
- 成交量越大，赚得越多
- Gas-free 让高频报价可行

**风险**：
- **库存风险**：单边成交导致持仓累积
- **趋势风险**：趋势行情中逆势接盘（同 HLP）
- **竞争**：其他做市商压缩价差

### 2.3 适合的市场

- 高流动性、震荡的市场（价差稳定）
- 不适合：单边趋势强烈的市场

---

## 3. 资金费率套利

### 3.1 基本逻辑

```
当资金费率很高（比如 0.1%/8h）：
  1. 在 Hyperliquid 做空永续（收资金费）
  2. 在现货买入等量（对冲方向风险）
  3. 净收益 = 资金费率收入（方向风险被对冲）
```

### 3.2 收益与风险

**收益**：
- 资金费率高时年化可达 30-100%
- 方向中性（不赌涨跌）

**风险**：
- **资金费率反转**：费率突然变负 → 反而付费
- **现货端成本**：需要现货持仓（资金占用）
- **执行风险**：两边不同步可能产生敞口

### 3.3 为什么这是相对稳的策略

```
方向被对冲 → 不怕涨跌
只赚资金费率 → 收益来源明确
风险主要是 "费率反转" → 可以监控和退出
```

---

## 4. Delta 中性策略

### 4.1 基本逻辑

```
Delta = 仓位对价格变化的敏感度
Delta 中性 = 总 Delta ≈ 0 = 不受价格涨跌影响

例：做多 1 BTC 永续 + 做空 1 BTC 永续（不同市场/期限）
→ 价格涨跌不影响净值
→ 赚的是两者之间的价差/基差
```

### 4.2 收益与风险

**收益**：
- 稳定（不赌方向）
- 来自基差、资金费率、价差

**风险**：
- 执行复杂（需要精确对冲）
- 基差风险（对冲不完美）
- 极端行情下对冲可能失效

---

## 5. 跟单策略（Trader Vault）

### 5.1 基本逻辑

```
某交易员历史收益好
你存钱到他的 Vault
他怎么交易，你的钱跟着怎么动
他赚 → 你赚（扣除他的抽佣）
```

### 5.2 收益与风险

**收益**：
- 如果交易员真的厉害 → 可观
- 省心（不用自己操作）

**风险**：
- **交易员失误**：他亏 → 你亏
- **幸存者偏差**：你看到的是"赢家"，输家不会展示
- **风格突变**：交易员可能突然激进
- **道德风险**：交易员用你的钱冒险（反正不是他的）

### 5.3 跟单的陷阱

```
看到一个 Vault 年化 200%：
  可能是真厉害
  也可能是 "赌对了一次"（高杠杆 + 运气）
  → 历史高收益 ≠ 未来高收益
  → 高收益往往伴随高风险
```

## 6. Vault 风险评估框架

### 6.1 评估一个 Vault 的清单

| 维度 | 看什么 | 警惕信号 |
|------|--------|---------|
| **历史收益曲线** | 是否平稳 | 暴涨暴跌 = 高杠杆赌博 |
| **最大回撤** | 历史最大亏损 | 回撤 > 30% 要警惕 |
| **持仓时间** | 运营多久 | 太新没经历过极端行情 |
| **当前持仓** | 杠杆、集中度 | 高杠杆、单一仓位 = 高风险 |
| **策略类型** | 是否可理解 | 看不懂的策略别投 |
| **规模** | TVL | 太小可能不稳定 |
| **抽佣** | 管理费/绩效费 | 过高侵蚀收益 |

### 6.2 收益的"质量"

```
高质量收益：
  来自做市价差、套利、稳定策略
  收益曲线平稳
  可持续

低质量收益：
  来自高杠杆方向押注
  收益曲线大起大落
  一次黑天鹅清零
```

### 6.3 红旗信号

```
🚩 年化 > 100% 且声称 "低风险" → 几乎一定是骗局或高风险
🚩 收益曲线只涨不跌 → 可能在累积尾部风险
🚩 策略不透明/看不懂 → 不要投
🚩 管理者匿名且无历史 → 风险高
🚩 大量营销/喊单 → 警惕
```

---

## 7. Vault 的系统性风险

### 7.1 Vault 之间的关联

```
如果很多 Vault 用相似策略（比如都做市）：
  极端行情下 → 它们同时承压
  → 系统性风险
```

### 7.2 大 Vault 被狙击

```
大 Vault 的持仓链上可见：
  攻击者可以针对性操纵
  → JELLY 类攻击（针对 HLP）
```

### 7.3 流动性风险

```
大量用户同时赎回一个 Vault：
  Vault 被迫平仓
  → 影响市场价格
  → 影响其他存款人
```

---

## 8. 对用户的总结建议

### 8.1 投 Vault 前问自己

1. 我理解这个策略吗？
2. 历史最大回撤是多少？我能承受吗？
3. 收益来源是什么？可持续吗？
4. 管理者是谁？有历史记录吗？
5. 我投入的钱占我总资产的比例合理吗？

### 8.2 分散原则

```
不要把所有钱放一个 Vault
不要只投高收益 Vault
保留一部分不投 Vault（流动性）
```

### 8.3 心态

```
Vault 收益 = 承担某种风险的回报
没有 "无风险高收益"
透明 ≠ 安全（你能看到风险，不代表风险消失）
```

---

## 9. 小结

1. **策略类型**：做市、资金费率套利、Delta 中性、趋势、跟单、统计套利
2. **风险-收益谱**：Delta 中性最稳，跟单/趋势最博
3. **做市**：赚价差，风险是库存和趋势
4. **资金费率套利**：方向中性，风险是费率反转
5. **跟单**：省心但有道德风险和幸存者偏差
6. **评估框架**：看收益曲线、回撤、持仓、策略、规模、抽佣
7. **红旗**：高收益+低风险声称、只涨不跌、不透明
8. **核心原则**：理解策略、分散投资、透明 ≠ 安全
