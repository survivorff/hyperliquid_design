# Vault 系统：链上的"量化基金"

> Vault 让普通用户可以"存钱赚收益"，让策略师可以"用别人的钱做市"。这篇拆解 Vault 系统的设计和运作。

---

## 1. 什么是 Vault

### 1.1 基本概念

Vault 是一个**资金池 + 自动策略**的组合：

```
用户存入资金（USDC）
    │
    ▼
Vault 按某种策略自动交易（做市、套利、跟单等）
    │
    ▼
产生的盈亏归存款人（按份额分配）
    │
    ▼
策略管理者抽取一部分作为报酬
```

### 1.2 类比

| 传统金融 | Hyperliquid Vault |
|---------|-------------------|
| 对冲基金 | 策略 Vault |
| 跟单交易（eToro） | Trader Vault |
| 做市商 | Market Maker Vault |
| 货币市场基金 | 低风险套利 Vault |

**核心区别**：Vault 是**链上、透明、无需信任**的。你能看到 Vault 的每一笔交易和持仓。

## 2. Vault 怎么运作

### 2.1 存款与份额

```
Vault 当前规模：$1,000,000，总份额 1,000,000
你存入 $10,000
→ 你获得 10,000 份额（占 1%）

Vault 通过交易增长到 $1,100,000
→ 你的 10,000 份额现在值 $11,000（+10%）
```

份额机制保证公平：后来者不会稀释先来者的收益。

### 2.2 收益分配

```
Vault 盈利
    │
    ▼
策略管理者抽佣（如 10% 的利润）
    │
    ▼
剩余按份额分配给存款人
```

### 2.3 提款

- 用户可以赎回份额，取回对应价值的 USDC
- 可能有锁定期（防止快进快出影响策略）
- 大额赎回可能需要 Vault 平仓（影响其他存款人）

## 3. Vault 的类型

### 3.1 官方 Vault：HLP

- Hyperliquid 官方运营
- 全市场做市 + 清算接盘
- 最大、最核心的 Vault
- 详见 [02-hlp-deep-dive.md](./02-hlp-deep-dive.md)

### 3.2 用户 Vault（Trader Vault）

- 任何人可以创建
- 公开自己的交易策略
- 其他人可以"跟投"（存钱进去）
- 管理者抽取利润分成

```
某交易员历史收益好
    │
    ▼
创建 Vault，公开交易
    │
    ▼
粉丝存钱进去 "跟单"
    │
    ▼
交易员赚钱 → 粉丝分成 → 交易员抽佣
```

### 3.3 协议 Vault

- HyperEVM 上的 DeFi 协议运营的 Vault
- 比如收益聚合器、结构化产品
- 可能组合多种策略

## 4. Vault vs GMX 的 GLP

很多人拿 Vault 和 GMX 的 GLP 对比。它们有本质区别：

| 维度 | GMX GLP | Hyperliquid Vault |
|------|---------|-------------------|
| 角色 | 被动 LP（交易对手方） | 主动策略（做市/套利） |
| 定价 | Oracle | 订单簿撮合 |
| 收益来源 | 交易者亏损 + 手续费 | 做市价差 + 手续费 + 策略 |
| 风险 | 交易者赚钱时 LP 亏 | 策略失败时亏 |
| 主动性 | 被动 | 主动 |
| 透明度 | 池子组成可见 | 每笔交易可见 |

### 4.1 关键区别：被动 vs 主动

```
GLP（被动）：
  GLP 是交易者的对手方
  交易者赚 = GLP 亏（零和）
  GLP 不"操作"，只是被动承接

Vault（主动）：
  Vault 主动做市、套利
  赚的是价差和手续费，不是"赌交易者亏"
  Vault 有策略，会主动调整
```

这个区别很重要：GLP 的收益和"交易者整体盈亏"绑定（交易者赢则 GLP 亏），而 Vault 的收益来自做市价差（理论上和交易者输赢无关）。

## 5. Vault 的透明度

### 5.1 链上可见

```
你可以查看一个 Vault 的：
- 当前持仓（多空、数量）
- 历史交易（每一笔）
- 历史收益曲线
- 当前规模和份额
- 存款人数量
```

### 5.2 透明度的双刃剑

**好处**：
- 你能判断一个 Vault 的策略是否靠谱
- 无法造假收益（链上数据）
- 可以及时发现异常

**坏处**：
- 策略被公开 → 可能被抄袭或针对
- 大 Vault 的持仓被看到 → 可能被狙击（JELLY 类）

## 6. Vault 生态的繁荣

### 6.1 为什么 Vault 在 Hyperliquid 特别活跃

```
1. 链上策略可以原子组合（HLP 的清算接盘是例子）
2. Gas-free 订单让高频做市策略可行
3. 透明度让用户敢于"跟投"
4. 没有 CEX 那种 "黑箱基金" 的信任问题
```

### 6.2 规模

- HLP 是最大的 Vault（数亿美元规模）
- 数百个用户 Vault
- 总 Vault TVL $500M+

---

## 7. 小结

1. **Vault = 资金池 + 自动策略**，存款人分享收益，管理者抽佣
2. **份额机制**保证公平分配
3. **三种类型**：官方（HLP）、用户（Trader Vault）、协议 Vault
4. **vs GLP**：Vault 是主动策略，GLP 是被动对手方
5. **透明度**是核心优势，但也是双刃剑（策略可能被针对）
6. **Hyperliquid 的 Vault 特别繁荣**：原子组合 + Gas-free + 透明
